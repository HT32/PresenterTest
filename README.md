# Presenter Test

* MVPアーキテクチャの勉強用プロジェクト

## ◆ 目的
* Presenterの実装方法について手を動かして勉強してみる。

## ◆ 目標
* Presenterの実装方法の習得。
* Presenterを利用することのメリットを見出す。

## 実装内容
* Fragmentにタイトル表示用のTextView, タイトル文字列表示アクション用のButtonを配置。
* Buttonを押下することでタイトル部にTextを表示する。
